Our root is html element which has font-size: 10px so our REM is equal to 10px

Purple element has width 1rem so it means - to calculate how many px it is in real we have to do operation: 1rem = 1 * REM = 1 * 10px - in this case our root element is html, html has font-size: 10px

Red element has width 1em so it means - to calculate how many px it is in real we have to do operation: 1em = 1 * EM = 1 * 10px - in this case our parent element is body, body has font-size: 10px
Gray element has diffrient value for font-size: 35px and width: 80%

Pink element has width 1rem so it means - to calculate how many px it is in real we have to do operation: 1rem = 1 * REM = 1 * 10px - in this case our root still is element html, html has font-size: 10px

Blue element has width 1em so it means - to calculate how many px it is in real we have to do operation: 1em = 1 * EM = 1 * 35px - in this case our parent element is gray div element, which has font-size: 35px

Green element has width 1em and diffrient font-size: 20px so it means - to calculate how many px it is in real we have to do operation: 1em = 1 * EM = 1 * 35px - in this case our parent element is gray div element, which has font-size: 35px - NOT THIS TIME - if current element has own font-size it has higher priority then parent so it means: 1em = 1 * EM = 1 * 20px

Aqua element has width 50% so it means it is 50% of width of our parent (grey element)

Brown element has width 50vw so it means it is 50 units of viewport width (viewport - the user's visible area of a web page (0-100 range))
